# Programmer_Raize_RPG
프로그래머 키우기 IP 기반 TRPG 형식의 게임

 어느날, RPG 세계에 소환 된 3류 프로그래머. 
 현실로 돌아가기 위해 RPG 세계에 신 '루비' 와 내기를 하는데.. 
 현실로 돌아가기 위해 바이러스를 때려 잡는 3류 프로그래머의 모험이 시작 된다!
 
 =플레이 방법=
 
 1. exe파일을 실행 시킵니다.
 2. 's' 를 입력하여 초반 튜토리얼을 진행합니다.
 (2-1). 'p' 를 입력하여 초반 스토리를 스킵할 수 있습니다.
 3. 닉네임을 입력합니다.
 4. 메인 화면으로 이동 합니다.
 5. 메인화면에서 '1'을 입력하여 스토리를 확인하실 수 있습니다.
 6. 메인화면에서 '2'를 입력하여 던전에 입장 하실 수 있습니다.
 (6-1). 던전 입장 전에 '필드'에서 원하는 몹(바이러스)을 선택하실 수 있습니다.
 7. 선택한 몹과 플레이어가 등장 합니다.
 8. 기본공격, 1번스킬, 2번스킬, 3번스킬을 통해서 몹을 처치하실 수 있습니다.
 9. 메인 화면에서 '3'을 입력하여 상점에 입장하실 수 있습니다.
 10. 플레이어의 '코어'가 0개가 되면 게임은 종료 됩니다.
